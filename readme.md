## Percent Calculator

Basic helper functions for percentage based calculations

### as()

What is `num1` as a percentage of `num2`

What is `20` as a percentage of `80`

```
percent.as(20, 80) // 25.00
```

Params:
* num1
* num2
* decimal (default 2)

### of()

What is `num1`% of `num2`

What is `23`% of `62`

```
percent.of(23, 62) // 14.26
```

Params:
* num1
* num2
* decimal (default 2)
